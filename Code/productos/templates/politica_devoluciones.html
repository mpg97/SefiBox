Política de devoluciones


Debido a las características de los productos y servicios vendidos en SefiBox, esta web no cuenta con la opción de realizar devoluciones de las compras que se realicen ni las suscripciones canjeadas.

Deberá comprobar y asegurarse de la compra antes de realizar el pago, ya que si este se efectúa será definitivo y no podrá devolver la suscripción comprada, asi como el importe cobrado en el método de pago facilitado para la compra.
